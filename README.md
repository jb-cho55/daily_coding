## 🗂️ 폴더 구조

이 레포는 **10문제 단위로 묶어 관리**합니다.

### 📁 관리 규칙

1. 문제는 **10문제 단위 묶음 폴더**로 관리
2. 묶음 폴더 이름은 `시작번호_끝번호` 형식
3. 각 문제는 **번호 폴더(001, 002 …)**로 관리
4. 문제 폴더 내부에 `cpp` 소스와 `exe` 실행 파일 포함

## 🧠 문제 풀이 기록 방식

```cpp
/*
[Problem] 0xx (백준 문제 번호)

- 핵심 접근 방법
- 사용한 알고리즘 패턴

[Complexity]
- Time: O(...)
- Space: O(...)

[Notes]
- 실수했던 포인트
- 주의할 엣지 케이스

# Do it! 알고리즘 코딩테스트 (C++) — 100문제 풀이

“Do it! 알고리즘 코딩테스트 C++편”에 수록된 **100문제**를 직접 풀고, 풀이 코드와 학습 기록을 정리한 레포지토리입니다.  
목표는 **기초 알고리즘/자료구조 체력 + 코딩테스트 실전 감각**을 만드는 것입니다.

> ✅ 사용 언어: C++17 (기본)  
> ✅ 문제 지문은 저작권 보호 대상일 수 있어 레포에는 포함하지 않습니다.  
> (각 파일에는 *핵심 아이디어/키워드/링크(가능한 경우)* 정도만 기록)
